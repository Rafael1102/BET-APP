## 🎯 Objetivo do Projeto
Este aplicativo foi desenvolvido para auxiliar técnicos, estudantes e comunidades no dimensionamento preciso de **Bacias de Evapotranspiração**. O foco é simplificar a coleta de dados e o cálculo de materiais necessários para a construção, promovendo soluções de saneamento descentralizado e sustentável.

## 🚀 Funcionalidades Principais
- **Cálculo de Materiais:** Dimensionamento automático de tijolos, cimento, areia e brita com base no número de moradores.
- **Relatórios em PDF:** Geração de relatórios técnicos detalhados prontos para impressão.
- **Partilha Facilitada:** Envio da lista de materiais diretamente para o WhatsApp.
- **Modo Offline:** Funciona sem internet, ideal para áreas rurais e comunidades isoladas.

## 🛠️ Técnicas e Tecnologias Utilizadas
O projeto utiliza as melhores práticas de desenvolvimento mobile moderno:
- **Flutter & Dart:** Framework e linguagem para alta performance.
- **Persistência de Dados:** Uso de `shared_preferences` para memorizar o estado da pesquisa do utilizador.
- **Navegação Nativa:** Implementação de `url_launcher` para integração com links externos.
- **Geração de Documentos:** Bibliotecas `printing` e `pdf` para criação de relatórios dinâmicos.
- **Avaliação de Usabilidade:** Implementação da metodologia **SUS (System Usability Scale)** integrada ao fluxo do app para validação científica (TCC).
